### Install
1. git clone https://github.com/nmaslov255/PyQT-WebKit-Loadfile.git
2. cd PyQT-WebKit-Loadfile
3. python3 -m venv .
4. source bin/activate
5. pip3 install -r requirements.txt
6. python3 main.py
